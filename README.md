# amaznt-backend
A backend project for a community where users share, browse, and discuss failed or hilariously bad purchases.
Rubbish Score - how rubbish your purchase was.

- Added Alembic migrations to keep track of schema changes.
- Created a User model with common fields like username, email, role, bio, created_at, and is_active.
